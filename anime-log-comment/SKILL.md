# Bangumi Anime Log Comment

进入本子 skill 前，先读：
- `../references/general-writing-layer.md`
- `references/writing-style.md`

## 适用任务

- 动画总评
- 单季 / 剧场版长评
- 单集延伸日志
- 从零散观感整理成站内长文

## 默认写法

1. 开头先给总判断，不先复述剧情
2. 第一部分抓住作品最能成立的一点：主题、角色关系、演出策略、节奏控制或情绪组织
3. 第二部分展开论证：角色、叙事结构、镜头 / 演出 / 配乐 / 节奏怎样支撑这个判断
4. 第三部分回到个人观感与边界：哪里打动你，哪里不成立，留了什么余韵
5. 结尾收回到“这部动画最后留下了什么”

## 输出要求

- 保留观感类语言，但不要碎到像弹幕整理
- 少做剧情流水账，多写“为什么这点有效 / 为什么无效”
- 如果是单集日志，可以更短，但仍要有清晰判断和展开
