# Bangumi Book Log Comment

进入本子 skill 前，先读：
- `../references/general-writing-layer.md`
- `../data/book/summary.md`
- `references/writing-style.md`

## 适用任务

- 小说 / 漫画 / 轻小说 / 学术向图书日志
- 读后感重写
- 长篇人物 / 主题分析
- 单卷 / 单章感想延展

## 默认写法

1. 开头先抛出读后判断或阅读后的核心不适 / 震动 / 喜爱点
2. 第一部分先落文本气质、主题或叙述方式
3. 第二部分再展开到人物、章节、结构、意象或作者处理手法
4. 第三部分处理个人阅读经验、局限、适读边界与余味
5. 结尾回到“这本书为何值得被记住 / 为什么我不认同它”

## 输出要求

- 允许更强的文本分析和概念密度，但不能只剩术语堆砌
- 可以用小标题梳理问题，但每节都要推进判断
- 不要把简介、设定、剧情梗概误写成评论主体
