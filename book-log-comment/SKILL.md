# Bangumi Book Log Comment

进入本子 skill 前，先读：
- `../references/general-writing-layer.md`
- `references/writing-style.md`

## 适用任务

- 小说 / 漫画 / 轻小说 / 学术书日志
- 读后感重写
- 长篇人物 / 主题分析
- 单卷 / 单章感想延展
- 剧情梳理与章节分析
- 人物线 / 意象线 / 叙述策略分析

## 默认写法

1. 先判断类型：总评 / 章节分析 / 人物分析 / 剧情整理
2. 素材不足时可查：作品简介、章节范围、角色信息、公开剧情摘要
3. 开头直接抛读后判断，或交代本篇要处理的剧情 / 章节范围
4. 第一部分先定文本气质、主题或叙述方式
5. 第二部分展开到人物、章节、结构、意象或作者手法
6. 第三部分处理个人阅读经验、局限、适读边界与余味
7. 结尾回到“这本书为何值得被记住”

## 输出要求

- 允许更高的文本分析密度，但不能堆术语
- 可用小标题梳理问题，但每节都要推进判断
- 不把简介、设定、剧情梗概当评论主体
- 只有简介级素材时，只写简介级分析
