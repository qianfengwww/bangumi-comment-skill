# Bangumi Game Log Comment

进入本子 skill 前，先读：
- `../references/general-writing-layer.md`
- `references/writing-style.md`

## 适用任务

- 通关日志
- 路线 / 周目感想
- 玩法系统评价
- 视觉小说 / AVG / CRPG / 动作游戏长评

## 默认写法

1. 开头先给通关后的总判断或最强烈的体验
2. 第一部分抓玩法、系统、文本、路线结构里最核心的成立点或问题点
3. 第二部分展开到流程节奏、难度曲线、角色 / 路线 / 系统反馈
4. 第三部分回看整体：值不值得推荐，适合谁，不适合谁
5. 结尾收束到“这段游玩经历最终留下了什么”

## 输出要求

- 可以保留更强的个人游玩记录感，但不能写成纯攻略
- 系统、关卡、路线、文本要和主判断发生关系，不要各说各话
- 若涉及剧透，先提醒再展开
